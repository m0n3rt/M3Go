# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-17
### Added
- Initial public release of Warrior Rimer.
- Core gameplay: movement, shooting, weapon switching (1/2/3), attack speed boost (F), pause (M).
- Achievement system with JSON data storage.
- Game data persistence (`game_data.json` & clean template).
- Packaging scripts: PyInstaller spec and optional Inno Setup script.
- Chinese & English README files.
- MIT License.
- Basic test skeleton using `unittest` (`tests/`).

### Planned
- Enemy AI improvements, difficulty progression, new weapons, configuration file, multi-language support.
